# $ や \_ を変数名として利用するライブラリ

* JQuery  
https://job-support.ne.jp/blog/javascript/howto-jquery  
jQueryの基本構文は以下のとおりです。  
`$("セレクタ").メソッド("パラメータ[引数]");`  
指定したセレクタに対して、メソッドを実行していくのがjQueryを扱う上での一般的な流れです。引数についてはメソッドの種類によって必要ない場合もあります。
